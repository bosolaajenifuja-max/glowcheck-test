# GlowCheck - Build & Launch Action Plan

*Last updated: 12th March 2026*

---

## Phase 1: Foundation (Weeks 1-4)

### 1. App Development Stack
- [ ] **Choose Framework:**
  - **Option A: React Native** — Cross-platform (iOS + Android), single codebase, popular for startups
  - **Option B: Flutter** — Cross-platform, good performance, slightly steeper learning curve
  - **Option C: Native (Swift/Kotlin)** — Best performance, but double the development time

- [ ] **Backend Setup:**
  - Database: Firebase (easiest) or Supabase (open-source alternative)
  - Authentication: Firebase Auth or Supabase Auth
  - API: Need to fetch product/ingredient data

- [ ] **MVP Features (Launch):**
  - Barcode scanning
  - Product search & ratings
  - Basic hazard analysis (the 13 categories)
  - Terms of Service + Privacy Policy

### 2. Product Database
- [ ] **Data Sources:**
  - EWG Skin Deep API (paid/freemium)
  - Open Food Facts (free, ingredients database)
  - CosIng (EU cosmetics database)
  - Manual community submissions

- [ ] **Data Strategy:**
  - Start with 500-1,000 top products in target categories (Black hair care)
  - Prioritise: Shampoos, conditioners, styling products, creams
  - Community submissions can grow database organically

- [ ] **Data Entry:**
  - Scraping existing databases (with permission)
  - Manual entry for high-priority products
  - Users can submit new products (verified by community/API)

## Phase 2: Legal & Compliance (Weeks 2-6)

### Must-Haves Before Launch
- [ ] **Terms of Service** — Finalise, add company details, solicitor review
- [ ] **Privacy Policy** — GDPR compliant, add to app + website
- [ ] **ICO Registration** — Check if required (likely yes if storing user data)
- [ ] **Disclaimer** — "Not medical advice" prominent in app

### Nice-to-Have
- [ ] **Privacy Policy on website** — Even if just a landing page
- [ ] **Contact email** — Dedicated support@ email
- [ ] **Cookie Policy** — If using analytics

### Insurance
- [ ] **Product Liability Insurance** — Protect against claims
- [ ] **Professional Indemnity** — Optional but recommended

---

## Phase 3: App Store Preparation (Weeks 4-8)

### Apple App Store
- [ ] **Apple Developer Account** — $99/year (individual or company)
- [ ] **App Store Guidelines Compliance:**
  - Remove any "medical advice" language
  - Clear privacy nutrition labels
  - In-app purchase setup (if monetising)
  - Age rating (likely 4+)
- [ ] **App Preview Screenshots** — Design 5-6 premium screenshots
- [ ] **App Preview Video** — 15-30 second demo
- [ ] **Submit for Review** — 24-48 hour review time

### Google Play Store
- [ ] **Google Play Developer Account** — $25 one-time
- [ ] **App Listing:**
  - Title, description, screenshots
  - Privacy policy URL required
  - Content rating questionnaire
- [ ] **App Signing** — Google manages keys
- [ ] **Submit for Review** — Usually faster than Apple

## Phase 4: User Growth Strategy (Launch +)

### Pre-Launch (Weeks 4-8)
- [ ] **Build Waitlist:**
  - Landing page with email capture
  - Social media teaser campaigns
  - Influencer early access

- [ ] **Community Building:**
  - Instagram, TikTok, YouTube content
  - Hair care forums (NaturallyCurly, Black Hair Lab)
  - Reddit communities (r/curlyhair, r/blackhair)

### Launch
- [ ] **Soft Launch:**
  - Limited release to waitlist
  - Gather feedback, fix bugs
  - Social proof (testimonials)

- [ ] **Launch Push:**
  - Press release
  - Influencer partnerships (Black hair care YouTubers, beauty bloggers)
  - Product hunt / app directory submissions
  - Reddit launch

### Post-Launch Growth
- [ ] **User Acquisition:**
  - ASO (App Store Optimisation) — keywords, screenshots, ratings
  - Paid ads (Meta, TikTok) — target demos
  - Referral programme — "Invite a friend"

- [ ] **Retention:**
  - Push notifications (new product scans, community updates)
  - Weekly safety tips
  - Loyalty programme (community contributions)

- [ ] **Monetisation (Optional):**
  - Freemium (pay for advanced features)
  - Premium subscription (ad-free, early access)
  - Affiliate links (product recommendations)
  - Brand partnerships (sponsored listings)
